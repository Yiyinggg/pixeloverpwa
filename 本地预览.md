# 本地预览最新修改

修改代码后想立刻看到效果，按下面方式操作。

## 方式一：用本地服务器（推荐）

**重要：必须从 pixeloverpwa 目录启动**，与 GitHub Pages 部署使用同一套文件。

在终端执行：

```bash
cd /Users/zhuyiying/Desktop/pixel-love-pwa/pixeloverpwa
npx serve .
```

然后打开浏览器访问：**http://localhost:3000**

每次改完代码，按 **Cmd+Shift+R** 强制刷新。若底部四个按钮显示 **森林、草地、海边、室内** 并带图标，说明加载的是最新版本。

---

## 方式二：用 GitHub Pages

1. 修改完成后执行：
   ```bash
   cd pixeloverpwa
   git add .
   git commit -m "描述本次修改"
   git push origin main
   ```

2. 等待 1–2 分钟，访问你的 GitHub Pages 地址（如 `https://yiyinggg.github.io/pixeloverpwa/`）

3. **强制刷新**（Cmd+Shift+R）或清除 Service Worker：
   - 打开开发者工具 (F12) → Application → Service Workers → Unregister
   - 刷新页面

---

## 版本标识

页面左上角有版本小徽章（如 **v22**）。若刷新后仍看不到变化，可清除 Service Worker：开发者工具 (F12) → Application → Service Workers → Unregister，然后刷新。
